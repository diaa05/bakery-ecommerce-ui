# bakery-ecommerce-ui
A visually crafted online bakery storefront developed using HTML and CSS, featuring an elegant product layout and smooth user interactions.

#Tech Stack
- HTML5
- CSS3
- Google Fonts
- Flexbox

#Purpose
This project was created to practice frontend web development, UI design, and layout structuring using pure HTML and CSS.

#Future Enhancements
- Add JavaScript-based cart functionality
- Implement category filtering
- Improve responsiveness using media queries
